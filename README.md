1.Gameplay 核心玩法实现：
为角色添加 Rigidbody2D 组件，通过监听玩家输入实现平滑移动与跳跃；设计 “区域触发式滑索” 核心机制（进入区域按 R 键禁用重力、沿路径移动），提升关卡交互性与玩法趣味性；
2.系统开发与体验优化:
用 Light2D + Mathf.Lerp 实现 3 分钟倒计时光源（范围 / 亮度随时间衰减），强化时间紧迫感；通过 LineRenderer + Raycast 做弹幕预警（红线 + 音效提示），提前告知玩家危险，优化操作体验；制作 3 层视差滚动背景（背景按不同速度移动）增强纵深感，搭建 UI 系统显示倒计时、血量等信息，用 AudioSource 管理背景 / 事件音效，保障交互流畅。
3.项目成果:
作品功能完整，玩法新颖，完成可玩Demo。
